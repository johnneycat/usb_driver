2、今早梳理时的发现：
	①每个驱动在加载后，只有一个实例（全局变量在程序开始前就会被创建和初始化）；
	②内核的智慧：虽然都采用了static方法去声明函数对别的文件隐藏函数，却通过函数指针暗度陈仓暴露接口
